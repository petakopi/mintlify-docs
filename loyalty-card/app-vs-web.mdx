# App vs web

The loyalty card system works across both the petakopi.my website and the mobile app. Some features are available on both platforms, while others are exclusive to one.

## For shop owners

| Feature | App | Web |
|---------|:---:|:---:|
| Create loyalty programs | ✓ | ✓ |
| Edit program settings | ✓ | ✓ |
| Activate/deactivate programs | ✓ | ✓ |
| Customize card colors | ✓ | ✓ |
| Wallet card preview | ✓ | ✓ |
| Scan customer QR codes | ✓ | — |
| Award stamps | ✓ | — |
| Redeem completed cards | ✓ | — |
| View enrolled customers | ✓ | ✓ |
| View activity log | ✓ | ✓ |
| View stamp history per card | ✓ | ✓ |
| Send messages to cardholders | ✓ | ✓ |
| View message history | ✓ | ✓ |

<Note>Scanning and stamping are only available on the mobile app. The website is for managing programs and viewing data — all point-of-sale actions require the app.</Note>

## For customers

| Feature | App | Web |
|---------|:---:|:---:|
| Get a loyalty card | ✓ | ✓ |
| Choose from multiple programs | ✓ | ✓ |
| Add to Apple Wallet (iOS) | ✓ | ✓ |
| Add to Google Wallet (Android) | ✓ | ✓ |
| View all my cards | ✓ | — |
| See stamp progress | ✓ | — |
| View stamp history | ✓ | — |
| Show QR code for scanning | ✓ | — |
| Remove a card | ✓ | — |
| Real-time wallet notifications | ✓ | — |

<Note>Customers can get their loyalty card on the website and add it to their phone wallet, but viewing cards and stamp progress requires the mobile app.</Note>
