---
title: "App vs Web"
description: "Compare loyalty card features available on the mobile app and website"
---

The loyalty card system works across both the petakopi.my website and the mobile app. Here's what's available on each platform.

## For shop owners

| Feature | App | Web |
|---------|:---:|:---:|
| Create loyalty programs | ✓ | ✓ |
| Edit program settings | ✓ | ✓ |
| Activate/deactivate programs | ✓ | ✓ |
| Customize card colors | ✓ | ✓ |
| Wallet card preview | ✓ | ✓ |
| Scan customer QR codes | ✓ | ✓ |
| Upload QR image | ✓ | ✓ |
| Award stamps | ✓ | ✓ |
| Redeem completed cards | ✓ | ✓ |
| View enrolled customers | ✓ | ✓ |
| View activity log | ✓ | ✓ |
| View stamp history per card | ✓ | ✓ |
| Send messages to cardholders | ✓ | ✓ |
| View message history | ✓ | ✓ |

<Tip>Both the web and app scanners support **uploading a QR image** — useful when a customer sends a screenshot of their QR code.</Tip>

## For customers

| Feature | App | Web |
|---------|:---:|:---:|
| Get a loyalty card | ✓ | ✓ |
| Choose from multiple programs | ✓ | ✓ |
| Add to Apple Wallet (iOS) | ✓ | ✓ |
| Add to Google Wallet (Android) | ✓ | ✓ |
| View all my cards | ✓ | — |
| See stamp progress | ✓ | — |
| View stamp history | ✓ | — |
| Show QR code for scanning | ✓ | — |
| Remove a card | ✓ | — |
| Real-time wallet notifications | ✓ | — |

<Note>Customers can get their loyalty card on the website and add it to their phone wallet, but viewing cards and stamp progress requires the mobile app.</Note>
