---
title: "App vs Web"
description: "Which owner features are available on the mobile app, the website, or both"
---

You can manage your coffee shop from both the petakopi.my website and the mobile app. Most basic shop-management features are available on both; the heavier analytical surfaces (Costs & Pricing, Pulse) are web-only.

## Shop management

| Feature | App | Web |
|---------|:---:|:---:|
| Edit shop name and description | ✓ | ✓ |
| Upload/remove logo | ✓ | ✓ |
| Upload/remove cover photo | ✓ | ✓ |
| Edit social media links | ✓ | ✓ |
| Select tags | ✓ | ✓ |
| Rich text description | — | ✓ |
