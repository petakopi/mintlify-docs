# Customer feedback

Customers can submit feedback about your shop through petakopi.my. View and manage all feedback from the **Feedbacks** section in the Manage panel.

## Viewing feedback

Each feedback entry shows:

- **Customer** — Username and email, or "Anonymous" if not signed in
- **Message** — The feedback text
- **Date** — When the feedback was submitted
- **Read status** — Unread feedback is highlighted with a badge

## Managing feedback

- **Mark as read** — Tap on a feedback entry to mark it as read
- **Unread count** — The sidebar shows a badge with the number of unread feedbacks

<Note>Feedback is one-way — customers submit it and you read it. There is no reply feature. If you need to respond, reach out through the contact information the customer provided.</Note>
