# App vs web

You can manage your coffee shop from both the petakopi.my website and the mobile app. Most management features are available on both platforms, with a few exceptions.

## Shop management

| Feature | App | Web |
|---------|:---:|:---:|
| Edit shop name and description | ✓ | ✓ |
| Upload/remove logo | ✓ | ✓ |
| Upload/remove cover photo | ✓ | ✓ |
| Edit social media links | ✓ | ✓ |
| Select tags | ✓ | ✓ |
| Rich text description | — | ✓ |

## Location

| Feature | App | Web |
|---------|:---:|:---:|
| Google Places search | ✓ | ✓ |
| Link Google Business profile | ✓ | ✓ |
| Drag map pin to adjust location | ✓ | — |
| Enter coordinates manually | — | ✓ |
| Use device GPS location | ✓ | — |

## Online store

| Feature | App | Web |
|---------|:---:|:---:|
